# SQL Data Analysis Project

## Overview
This project demonstrates SQL-based data analysis using a sales dataset.
The analysis focuses on extracting insights using filtering and aggregation.

## Dataset
`sales_data.csv` contains transaction-level sales data including:
- order_id
- customer
- product
- quantity
- price
- order_date

## Key Queries
- Retrieve all sales records
- Calculate total revenue per product
- Analyze customer purchasing behavior
- Filter sales by date

## Tools
- SQL
- CSV dataset
- GitHub

## Files
- `sales_data.csv` → sales dataset
- `queries.sql` → SQL analysis queries

## Outcome
The project answers basic business questions and demonstrates practical SQL skills.
