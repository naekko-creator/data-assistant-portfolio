# Excel Analysis Summary

## Dataset
The dataset contains daily sales records with product categories and revenue.

## Analysis Performed
The following steps were performed using spreadsheet tools (Excel / Google Sheets):

1. Created a Pivot Table to summarize total revenue by product
2. Created a Pivot Table to summarize total quantity by category
3. Applied basic sorting to identify top-selling products

## Key Insights
- Laptop generated the highest total revenue
- Accessories had higher total quantity sold compared to Electronics
- Mouse was the most frequently sold item

- Microsoft Excel / Google Sheets
