# Excel Data Analysis Project

## Overview
This project demonstrates basic data analysis using spreadsheet tools.
The goal is to summarize sales data and extract simple business insights.

## Dataset
`sales_summary.csv` contains daily sales data including:
- date
- product
- category
- quantity
- revenue

## Analysis
The analysis was performed using Pivot Tables and basic spreadsheet features:
- Total revenue by product
- Total quantity sold by category
- Identification of top-selling products

## Key Insights
- Laptop generated the highest revenue
- Accessories category had the highest quantity sold
- Mouse was the most frequently sold product

## Tools
- Microsoft Excel / Google Sheets

## Files
- `sales_summary.csv` → raw sales data
- `analysis.md` → analysis summary and insights
